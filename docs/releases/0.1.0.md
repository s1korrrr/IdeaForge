# IdeaForge 0.1.0

IdeaForge 0.1.0 is the first planned public binary release of the native idea
capture and engineering-workspace clients. It includes the Mac, iPhone, and
Apple Watch apps, the shared Swift core, and the localhost community backend.

## Highlights

- Capture and review spoken ideas across Apple platforms.
- Keep the default local workflow useful without a hosted service.
- Export reviewable engineering packets from structured workspaces.
- Verify Mac updates through a signed Sparkle appcast.

## Boundaries

This release does not include a production hosted backend. Cloud AI,
production sync, billing, APNs delivery, and web account management require
separately operated infrastructure. The community backend is for localhost
development and personal evaluation; do not expose it directly to the public
internet.

The release remains a draft until the exact tag passes protected CI, Developer
ID signature checks, Apple notarization and stapling, Gatekeeper assessment,
artifact attestations, checksum verification, and a clean-Mac update test.

## Verification

Official artifacts will include a DMG, Sparkle update ZIP, checksums, manifest,
SPDX SBOM, signed appcast, sanitized notarization evidence, and GitHub artifact
attestations. Follow [the independent verification procedure](../RELEASING.md#independent-verification).
